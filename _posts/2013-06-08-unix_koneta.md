コマンドライン叩いてるとき、「え？今の何？」って良く言われるのが、

```
$ cp hoge{,.bak}
```

みたいなやつ。

一応解説しとくと、シェルが、

```
$ cp hoge hoge.bak
```

に展開してくれる。バックアップファイル作ったり、ファイル名 typo したのを直したりするのにわりと良く使う。
tcsh と bash で使える。多分 zsh も使えるんじゃないかな。

こういうので、知ってるけど普段意識してないのものとか、知らないものとか、色々ありそうだなー、って
久しぶりに「今のなんすか？」って聞かれたときに思ったのでした。
